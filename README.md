# Technavyug Frontend

A modern React + Vite frontend for the Technavyug platform featuring:

* Responsive UI
* Admin Dashboard Integration
* Projects Showcase
* Certificate Verification
* Achievements Management
* Events & Media Sections
* Interactive User Experience

## Tech Stack

* React.js
* Vite
* Tailwind CSS
* React Router DOM
* Axios

## Features

* Dynamic Projects Display
* Certificate Verification Portal
* Achievement Showcase
* Admin Panel Integration
* Secure API Communication
* Responsive Design

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```env
VITE_API_URL=your_backend_url
```
